### N의 크기에 따른 허용 시간 복잡도 (대략적인)

| N의 크기      | 허용시간복잡도 |
| ------------- | -------------- |
| N<= 11        | O(N!)          |
| N <= 20       | O(2^N)         |
| N<= 100       | O(N^4)         |
| N<=500        | O(N^3)         |
| N<=3000       | O(N^2logN)     |
| N<=5000       | O(N^2)         |
| N<=1,000,000  | O(NlogN)       |
| N<=10,000,000 | O(N)           |
| 그 이상       | O(logN), O(1)  |
